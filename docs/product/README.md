# Smart Paper 任务文档索引

## 📚 文档概览

本目录包含 Smart Paper 智能错题本项目的完整产品规划和实施任务文档。

### 产品愿景

为小学生打造一个智能错题本系统，通过 AI 技术实现错题的智能收集、分析、整理和巩固学习，**让孩子掌握得更加牢固**。

## 🏗️ 技术栈

### 后端
- **框架**: FastAPI (Python)
- **OCR**: Qwen2.5-VL / SiliconFlow Vision API
- **AI 生成**: LLM (GPT-4/Claude/Qwen)
- **图像处理**: OpenCV / Pillow
- **数据库**: PostgreSQL / MySQL（待实施）
- **对象存储**: S3 / OSS / MinIO（待实施）

### 前端
- **框架**: React + Vite
- **UI 组件**: 待定
- **状态管理**: 待定
